# 结语

## 学到了什么？

1. 快速简单搭建Go开发环境
2. 导入包、申明包
3. func 定义函数
4. 变量的申明方法
5. Go语言的异常处理
6. for循环
7. map类型
8. 用http包，编写一个网站程序

本系列内容很少，很简洁，但是如果能让你对Go多一点点了解，让你对学习Go多增加一点点兴趣，也就达到本文的目的了。
